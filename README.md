# DB 프로젝트(Oracle) - 교육센터 관리시스템
- 개발목적: 교육센터 운영 및 유지관리 프로그램을 오라클을 이용하여 구현한다.
- 담당업무
  - 관리자
    - 교육생 등록 및 출결조회/관리, 시험 및 성적 관리
  - 교사
    - 계정관리, 강의스케줄 조회, 배점 및 성적 입출력, 교육생 출결 조회, 근태 관리

### 개발환경 및 사용기술
- 개발 플랫폼: Windows 10 64Bit
- 개발언어: Oracle SQL
- 개발도구: Eclipse, SQL Developer, eXERD
- 사용기술: DBMS, JDBC

### 핵심기능
- 관리자
  - 과목, 과정 개설 및 관리
  - 교사 및 교육생 정보 등록 및 관리
- 교사
  - 강의 스케줄 조회
  - 배점 및 성적 입력 및 출력
- 교육생
  - 출결 및 성적 조회
